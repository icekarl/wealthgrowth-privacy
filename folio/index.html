Core Features

为资产记录而设计，不是流水账本

财记 Folio 更关注“我现在拥有哪些资产、成本是多少、收益怎么变”，适合长期维护个人持仓。

统一记录多类资产

股票、基金、现金、理财、黄金、加密货币都可以放在一个组合里，按类型和标签查看。

自动计算成本与收益

按移动加权平均成本处理买入、卖出、送股和校准，减少手动维护均价的麻烦。

多币种与汇率换算

美元、港币、人民币等资产统一折算到主币种，外币现金和美股、港股、BTC 的汇率影响会被纳入参考收益。

基金与定投记录

支持基金净值、QDII 时间差、LOF 场内场外口径，以及按周期补记定投买入流水。

现金联动与分红

卖出股票、现金分红会回到对应币种的证券账户现金，帮助你核对资金进出。

本地优先，iCloud 同步

资产和交易记录保存在本机；Pro 用户可通过 Apple iCloud 私有数据库在同一 Apple ID 设备间同步。

常见问题

财记 Folio 是什么？

财记 Folio 是一款个人资产记录与组合追踪工具，帮你手动记录股票、基金、现金等资产的买卖与持仓，查看参考市值、收益走势和资金进出。

首页总资产是怎么计算的？

总资产等于每一项持仓的当前参考市值加总，并折算到你的主显示币种。单项市值通常由持有数量、最新价格和汇率共同决定。

成本价是我自己填的，还是 App 算的？

财记 Folio 会按移动加权平均成本自动计算。你每记录一笔买入，系统会把金额并入成本池重新摊平；卖出时按比例扣减。

美元现金为什么也会有收益？

因为外币资产会折算成主币种显示。美元金额没有变化时，只要美元兑人民币汇率变化，折算后的资产价值也会变化，因此会体现在收益里。

基金定投计划会替我下单吗？

不会。定投计划只是记账助手，用来按频率补记买入流水。财记 Folio 不参与任何真实资金交易，也不替你向基金平台提交订单。

我的数据会上传到财记 Folio 服务器吗？

你手动输入的资产、交易、现金账户、成本和备注保存在本机。Pro 用户启用 iCloud 同步时，数据存放在用户自己的 Apple iCloud 私有数据库中。
